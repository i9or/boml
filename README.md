# boml

Blue Oak Model License CLI generator for your repository.

![Running CLI Screenshot](/screenshot.png)

## Usage

For interactive step by step questionnaire run:
```bash
$ npx boml
```

To generate default license file:
```bash
$ npx boml -d
```

For help:
```bash
$ npx boml -h
```
